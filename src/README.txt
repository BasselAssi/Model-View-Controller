MVC
===

Exemple d'application suivant l'architecture MVC.


Le programme a deux points d'entrée, selon si on souhaite l'exécuter
en ligne de commande ou en interface graphique :

    java VueConsole
    java VueGUI


On peut facilement utiliser le modèle, Compteur, dans un autre projet.

On peut également remplacer le modèle Compteur par une autre classe
compatible, par exemple CompteurMoyenne, qui a les mêmes méthodes
publiques mais qui a un comportement différent.


==========================================================================================

MVC
Example of application following the Model View Controller architecture.

The program has two entry points, depending on whether you want to run it
from the command line or in a graphical interface:

    java VueConsole
    java VueGUI

The model, Counter, can easily be used in another project.

The Counter model can also be replaced by another compatible class,
for example AverageCounter, which has the same public methods but behaves differently.
